# Aqsa Riaz - Computer Science Portfolio

A responsive **static portfolio website** built with **HTML and CSS** as part of an HTML/CSS roadmap assignment.
This project demonstrates **semantic HTML**, **Flexbox**, **CSS Grid**, **responsive design**, and **relative vs absolute positioning** through a multi-section portfolio layout.

---

## Project Overview

This portfolio presents my academic background, technical skills, learning roadmap, and contact information in a clean multi-section layout.
The goal of this project is to practice front-end layout fundamentals by building a static webpage using only **HTML** and **CSS**.

---

## Features

* Semantic HTML structure using:

  * `header`
  * `nav`
  * `main`
  * `section`
  * `article`
  * `aside`
  * `address`
  * `footer`

* Responsive layout using **CSS Grid**

* Navigation and content alignment using **Flexbox**

* Demonstration of the **CSS box model**

* Use of **margin**, **padding**, **borders**, and **spacing**

* Example of **relative and absolute positioning**

* Multi-section static portfolio design
